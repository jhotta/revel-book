## ソースをコンパイルする手順

### 1. ReVIEWのインストール

下記を参考にしてください。
https://github.com/kmuto/review

## ソースコードのコンパイル
src以下に配置された、re(拡張子)ファイルをreviewのCLIコマンドを使ってコンパイルしていきます。makeを実行したディレクトリーにpdfとepubファイルが生成されます。

```
$ make
```

## コンパイル結果の削除

pdfファイルとepubファイルを生成するために使用した一時ファイル及び、生成結果のファイルを削除します。

```
$ make clean
```
